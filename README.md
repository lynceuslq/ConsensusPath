## Introduction

### ConsensusPath is a python script that generate presumebly refined protein sequences from given MSA by idenfying consensus substibutions from the MSA and combining the substitutions while maintain the functional stability of the proteins with protein large language model ESM-C (you can find useful infomation at https://github.com/evolutionaryscale/esm/tree/main).
